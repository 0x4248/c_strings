# C Strings

The GNU strings command remade in C.

## Usage

```
$ strings [FILE] [OPTION]
```

## Options

```
-ln, --line-numbers     Show line numbers
-c, --context           Show context of each string
-h, --help              Show this help message
-v, --version           Show version
```

## Examples

```
strings /bin/ls
```
